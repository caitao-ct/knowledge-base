# 订单流程

## 业务流程

用户下单到订单完成的完整流程：

```
创建订单 → 支付 → 商家确认 → 发货 → 确认收货 → 完成
           ↓
      支付失败 → 可重试 (3 次)
```

## 状态流转

| 状态 | 说明 | 可操作 |
|------|------|--------|
| PENDING | 待支付 | 支付、取消 |
| PAID | 已支付 | 商家确认 |
| CONFIRMED | 已确认 | 发货 |
| SHIPPED | 已发货 | 确认收货 |
| COMPLETED | 已完成 | 可评价 |
| CANCELLED | 已取消 | 不可操作 |
| REFUNDING | 退款中 | 不可操作 |

## 业务规则

1. 订单超过 30 分钟未支付自动取消
2. 已支付订单取消需走退款流程
3. 同一用户最多同时有 5 个进行中的订单
4. 订单总金额超过 5000 元需人工审核
